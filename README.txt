Shoplifting Detection – Demo

Demo de detección de comportamiento sospechoso en video usando poses humanas.

Requisitos

Python 3.9+

Instalar dependencias:

pip install ultralytics torch opencv-python numpy

Cómo correr la demo

Desde la raíz del proyecto:

python src/infer_video.py --video data/raw_videos/demo_video.mp4


Presiona ESC para cerrar el video.
